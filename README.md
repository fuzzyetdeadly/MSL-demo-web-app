# My first web app (a toy project)

Access the app [here](https://msl-demo-web-app.onrender.com/)

A simple web-app, for the sole purpose of learning how to create web-apps with .NET/ASP.NET Core.
It is mostly created by following tutorials from `Microsoft Learn`.

Current features:

* A few simple endpoints
* A few simple tests
* Deployment to `render.com` as a POC for publishing to the cloud

To familiarize with web-app development, I intend to iteratively add more features as I learn more.
